## 大数相乘
* 2 个超大的数（比如 2 个 100 位的数），如何进行乘法？
   ```
   按照小学时学习的乘法去处，在进行 n 位数之间的相乘时，需要大约进行 n^2 次 个位数的相乘
   比如 36*54
   4 * 6 = 24
   4 * 30 = 120
   50 * 6 = 300
   50 * 30 = 1500
     24
    120
    300
   1500
   => 每一列相加
   1944
   ```
  
  
* 算法分析
   0. 题目
      ```
      一个数 AB 和 另一个数 CD 相乘
      AB * CD
      注意 AB 表示这个数由 前半A 和 后半B 组成，当然 CD 同理
      ```
   1. 分治
      ```
      A--B
      C--D
      计算过程：
      A-C
         B-D
       B-C
       A-D
      ```
   2. Karatsuba 算法
      ```
      BC + AD = AC + BD - (A-B)(C-D)
      BC + AD = AC+BD + (-AC+AD+BC-BD)

      A--B
      C--D
      计算过程：
        A-C
           B-D
         A-C
         B-D
      (A-B)(C-D)
      
      ```
